# 🤟 Silent SoulConnect Ministry (SSCM)

## Where Silence Speaks, and Hearts Connect

Silent SoulConnect Ministry (SSCM) is a Deaf-friendly Christian ministry dedicated to making faith, Bible study, prayer, encouragement, and community accessible to everyone.

Built with accessibility, simplicity, and spiritual growth in mind, SSCM provides daily devotions, Bible studies, prayer support, blogs, and outreach resources for Deaf, Hard of Hearing, hearing individuals, families, and faith communities around the world.

---

# 🌿 Our Mission

To connect hearts to Christ through accessible ministry, biblical truth, prayer, encouragement, and community.

### We Believe

* Silence is not weakness.
* Deafness is not absence.
* Every person matters to God.
* Faith should be accessible to all.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/SCWB2024/SilentSoulConnectMinistry.git
cd SilentSoulConnectMinistry
```

## Create a Virtual Environment

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python app.py
```

Open the local development server in your browser.
